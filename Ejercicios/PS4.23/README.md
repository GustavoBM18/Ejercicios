# PS 4.23 — Ingresos de alumnos en 7 carreras (12 años) — Java 21

## Descripción
Este programa analiza los datos de ingreso de alumnos en 7 carreras universitarias durante 12 años.
Proporciona los resultados:
a) Totales por año.
b) Porcentaje de alumnos en un año y carrera específica.
c) Año y carrera con menor ingreso.
d) Año en que una carrera tuvo su mayor ingreso.

## Instrucciones
```bash
javac -d out $(find src -name "*.java")
java -cp out edu.primersemestre.IngresosUniversidad
```

## Notas
- Se deben ingresar 12×7 valores enteros no negativos.
- Los años corresponden a filas y las carreras a columnas.
